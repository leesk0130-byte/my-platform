# 트래픽 확대 로드맵: 편리·실용 도구형 커뮤니티

**목표**: 검색 유입(SEO) → 실사용(도구/정보) → 재방문(저장/알림/구독)  
**기준**: LCP·모바일 가독성 우선, 기능은 작게 완성된 UX로 배포

---

## 1주일 우선순위 (3단계)

### 1단계 (즉시 배포 가능) — 검색·북마크·계산기 A/B·뉴스 운영
- **IA**: 헤더 메뉴 5개 고정 (커뮤니티, PG 수수료, 수수료 계산기, 최신 공지/뉴스, 인증/제휴 문의), 나머지(가맹점숲이란, 꼭 알아야 할 것) 푸터
- **홈 허브**: 검색창 1개(→ 커뮤니티 검색), 인기글 Top5 + 최신글 Top5, 계산기·PG 수수료 바로가기
- **커뮤니티**: 검색/필터/정렬 유지, **북마크(localStorage)** + **내가 저장한 글(/bookmarks)**, 글 상세 **관련 글 5개**
- **계산기**: **A vs B 비교** (공통 입력 + PG A/B 패널), **저장(이름 붙이기)** + **저장 목록(/calculator-saved)**, **공유 URL**(쿼리 인코딩/복원)
- **뉴스**: 운영자 공지/뉴스 시스템 유지(D1+Functions), 빈 상태 문구
- **정리**: 광고 placeholder 제거, KPI 이벤트 스텁(페이지뷰/검색/계산/북마크/공유, privacy 정합)

### 2단계 — SEO·고정 URL·내부링크
- 커뮤니티 글 상세 고정 URL: `/community.html?id=xxx` 또는 `community/post/xxx` (검색 노출)
- 각 페이지 title/description/canonical/OG 유지·점검, sitemap.xml·robots.txt
- PG 수수료 ↔ 계산기 ↔ 관련 커뮤니티 글로 교차 링크

### 3단계 — 재방문·가독성
- 글 상세: 긴 글이면 목차, 코드/표 가독성, 공유 버튼(링크 복사)
- 리스트: 댓글수·조회수·상대시간·카테고리 칩 강화
- (선택) 구독/알림, 이미지·폰트 최적화

---

## 핵심 UX 흐름

```
홈 → [검색: "PG 수수료/정산/오류/가이드"] → 커뮤니티 결과
   → 인기 Top5 / 최신 Top5 클릭 → 글 상세 → 북마크 저장 / 관련 글
   → [계산기 바로가기] → A vs B 입력 → 결과 저장/공유 링크
   → [PG 수수료 바로가기] → 가이드·표 → 계산기·관련 글 링크
```
