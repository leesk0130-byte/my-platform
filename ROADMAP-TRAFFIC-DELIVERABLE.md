## 2. 유입 퍼널 (검색 → 가이드 → 계산기 → 커뮤니티)

1. **검색 유입**  
   - 검색엔진: `must-know/*` 8개 문서, `pg.html`, `calculator.html`, `news.html` 등이 sitemap·메타로 노출.  
   - 사이트 내: 헤더·홈 검색 → `community.html?q=검색어`.

2. **가이드**  
   - `must-know.html` → 키워드별 가이드 8개 → 각 문서 하단 "커뮤니티에서 [키워드] 검색하기".

3. **계산기**  
   - `calculator.html` 결과 하단 "관련 가이드·커뮤니티" → must-know 문서·수수료 커뮤니티·저장한 계산 목록.  
   - 공유: 링크 공유·이미지 저장 → 재방문·공유 유도.

4. **커뮤니티**  
   - 질문 템플릿(정산지연/환불/가상계좌/심사) + 필수 정보(업종·월거래액·PG)로 질문 품질·검색 연계 강화.  
   - 글 상세 "관련 글 5개"로 이탈 감소·추가 읽기 유도.

## 3. "빈 상태일 때 섹션 숨김" 적용 여부 체크리스트

| 위치 | 적용 여부 | 구현 |
|------|-----------|------|
| 홈 — 최신 뉴스 미리보기 | ✅ | `index.html`: `fetchNews(5,0)` 후 `items.length === 0`이면 `#news-preview` `display: none` |
| 홈 — 인기글 Top5 | ⚠️ | 비어 있어도 섹션 노출 가능. 원하면 `items.length === 0`일 때 해당 section 숨김 추가 가능 |
| 홈 — 최신글 Top5 | ⚠️ | 동일 |
| 뉴스 페이지 — 리스트 | ✅ | `news-render.js`에서 EmptyState("등록된 뉴스가 없습니다") 표시, 리스트 없음 |
| 커뮤니티 — 관련 글 5개 | ✅ | 관련 글이 없으면 `#related-posts-section` `display: none` |
| 저장한 계산 목록 | ✅ | `calculator-saved.html`에서 목록 비면 EmptyState만 표시 |
